# PipsUUF
Adds noticeable pips to the empowered player castbar of Unhalted Unit Frames.

This WoW addon, provides a simple, lightweight method of adding more visible
pips to the player castbar of Unhalted Unit Frames.  No customization is
provided, other than editing the PipsUUF.lua file, as illustrated below.

```
local PIP_WIDTH = 2           -- Width of each pip
local PIP_LAST_WIDTH = 5      -- Width of the last pip
local PIP_COLOR = { 1, 1, 1 } -- Color of each pip
local PIP_ALPHA = 1           -- Alpha of each pip
```
